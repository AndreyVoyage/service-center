# ИТЕРАЦИЯ 2B: RBAC — Финальный Checkpoint

## ✅ Verification (Checkpoint 2)

### Шаг 1: Перезапустить `pnpm dev`

```bash
cd apps/cms
pkill -f "next dev" || true
pnpm dev
```

### Шаг 2: Тест Developer

1. Войти как `dev@service.center` (developer)
2. Проверить что видно меню Users
3. Проверить что можно создать пользователя

**Ожидаемый результат:**
- ✅ Видит коллекцию Users
- ✅ Может создать пользователя
- ✅ Может менять роли

### Шаг 3: Тест Admin

1. Открыть инкогнито-окно
2. Войти как `admin@service.center` (admin)
3. Проверить что видно

**Ожидаемый результат:**
- ❌ НЕ видит коллекцию Users
- ✅ Видит только свои данные в профиле

## 📋 Чек-лист (Итерация 2B)

- [ ] Перезапущен `pnpm dev`
- [ ] Developer видит коллекцию Users
- [ ] Developer может создать пользователя
- [ ] Admin НЕ видит коллекцию Users
- [ ] Admin видит только свои данные

## 🆘 Fallback Strategy

**Проблема:** Admin заблокирован
**Решение:** Вернуться к `create: () => true`, создать developer, вернуться к строгому режиму

**Проблема:** Не видит разделы
**Решение:** Проверить `access: { admin: ... }` возвращает boolean

**Если все галочки отмечены → переходим к ИТЕРАЦИИ 3 (Core Collections)**
